# XAIAK — RiskBrief AI (Prototype)

Structured AI-based risk intelligence engine.

This repository demonstrates:

- Converting unstructured text into structured risk insights
- Risk scoring and severity classification
- Escalation recommendation logic

Runs with local LLM (Ollama) or fallback rule engine.

## Concept

In AI-driven environments, unstructured instructions and messages can introduce risk.
This prototype shows structured risk analysis for governance-aware systems.
